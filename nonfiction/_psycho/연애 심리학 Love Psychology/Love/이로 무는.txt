


 

지금까지 윗 입술·눈·입안 등을 제외한 몸의 모든 부분에 입을
맞추는 것을 말하였으나 그 부분은 또한 상처를 내는 데나 이로 무는
데에도 역시 같다 할 수 있다. 이는 고르게 길게 ( 페데르 ) 의 잎을 씹는 것같이
남의 눈을 끄는 것이어야 하며, 크기는 알맞게 총총히 나서 벌어져서는 안 되며,
끝은 고르게 나와야 좋은 이라 할 수 있다.
반면에 이가 빠졌거나 갈라졌거나 고르지가 못한 이, 또는 길고
짧고 달아졌거나 사이가 벌어진 것, 그외 너무큰 이들은 모두 정상적인
상태의 이가 아니다. 이로 상처를 입히는 데에는
다음의 여덟 가지 종류가 있다.


1. 구드하캄
2. 웃쮸후나캄
3. 빈두
4. 빈두마라
5. 푸라바라마니
6. 마니마라
7. 칸다브흐라캄
8. 브라하쨔르비 타캄


여자의 아랫입술이 남자의 입술과 이와의 사이에 물려, 상처가
않나도록 조금만 물면 붉게 된다. 이것을 ( 구드하캄 ) 이라고 한다.
물린 자리가 나타나지 않으므로 비밀이라는 이름이 붙여지고 있는 것이다.
위에서 말한 행위가 보다 강열하게 물어서 그 자리가 부풀어 부어 있는 
것을 ( 웃쮸나캄 ) 이라고 한다.
위와 같은 행위는 ( 빈두 ) 와 같이 아랫입술의 중간을 물게된다.
( 웃쮸후나캄 ) 은 ( 푸라바라마니 ) 와 같이 뺨을 물게된다. 그러나 이는
왼쪽 뺨만을 물게된다. 꽃으로 아름답게 치장하거나 키스와 이로 물려서
생긴 상처는 부인의 왼쪽 뺨을 치장하게 된다. 여자의 몸의 어떤 중요한
위치에 상처가 나지 않을 정도로 이와 입술로 계속하여 물어서 붉게 피가
맺힌 점은 ( 푸라바라마니 ) , ( 푸라바라 ) 라고 한다.

이와 같은 수법으로 이런 종류의 상처가 계속많이 있는 것은
( 마니마라 * 화잠 ) 라고 한다. 여자 몸의 어떤 위치의 피부가 좀 부어
올랐거나 이로 물어서 피부에 작은 상처가 생기는 것을 ( 빈두 * 물방울 ) 라고 한다.
이런 종류의 점이 한 줄로 계속 있어서 같은 수법으로 몸의 어느 위치에 물린
것을 ( 빈두마라 * 물방울의 꽃술 ) 라고 한다.
이 두 방법의 ( 마라 * 꽃술 ) 는 목·겨드랑·사타구니 등을 물게된다. 
이 위치 등은 피부가 늘어지기 쉬우므로 이런 이름이 붙여진 것이다.
( 빈두마라 ) 는 또한 이마·넓적 다리도 물게된다. 구름과 같은 모양으로
이로 무는 것은 ( 칸다브흐라캄 ) 이라고 한다. 이럴때는 가슴 밑을 물게된다.
이곳은 꼭 문거같지 않기 때문이다. 강한 이로 물린 많은 흔적이 여자의
가슴 밑에 연달아 나타나고 그 중간 위치가 붉게 보이는 것을
( 브라하쨔르바타캄 ) 이라고 한다. 마치 멧돼지가 문자리와 같기 때문이다.
위에 말한 두 방법은 애인의 뜨거운 사랑의 표시로 물게된다.
이것으로 이로 무는 상처를 내는 행위에 관한 설명을 끝낸다.

손톱으로 할퀴어서 내는 상처와 이로 물어서 내는 상처는 애인에게
주는 회상의 선물이고 혹시 ( 비슈샤카 * 이마에 붙인 장식물의 나뭇잎 ) ,
( 카루나푸라 * 귀걸이 ) , ( 구슈파피디 * 꽃다발 ) , ( 담부라파라샤 * 페테르를
말한다 ) , ( 타마라 ) 의 잎 등에도 상처를 내는 일이 있다.
이 상처는 사랑을 나타내기 위한 예비적인 행위라는 뜻으로 ( 아브히요기카 )
라고 한다. 이런 애정 표시의 행위는 단순히 정해진 지방에서만 그들의
관습으로 전래해 오고있다. 
예를 들면 ( 맛갸데샤 * 야무나와 항하 / 恒河 ) 의 사이에 있는 지방)의
부인들은 순진한 아르야족으로서 습관도 매우 순결 하므로 키스나 손톱으로
내는 상처나 이로 무는 상처등은 모두 거절하고 만다.
( 바후리카 ) 및 ( 아반티 ) 의 부인들도 이와같은 행위를 전적으로 거절한다.
그러나 그들의 관습은 동작이 이상해서 그것도 기교적인 성교에 해당된다.
( 마라바 ) 및 ( 아브히라 ) 의 부인들은 주로 포옹·키스·상처가 나지 않을
정도의 할퀴기와 물기에 숙달되있다. 그러나 ( 프라하나 ) 라고 하여,
어떤 중요한 위치를 때림으로써, 최후의 만족한 쾌감을 느끼게 된다.

오하 ( 五河 ) 지방의 부인들은 원래 ( 아우파리슈타카 * 입으로
하는 성교 ) 에 숙달되었다. ( 아바란카타 ) , 즉 외부지방 서부해안에
가까이 사는 부인들과 ( 라다 ) 지방의 부인들은 정욕이 대단히 왕성하여
가벼운 충격에 능히 참고 ( 시트크리타 * 울부짖음 ) 을 연발하는 습관이 있다.
( 스트리라쥬야 * 서부 벵갈지방 ) , ( 코샤리지방 ) 에서는 부인들은 괴상한
성욕자들로 힘껏 때리는 행위를 즐긴다. 그리고 주로 ( 아파드라브야 )
남근모조탑를 애용한다.

( 나루마다 ) 강의 남부 ( 안드라데샤 * 데르그지방 ) 의 부인들은
성질이 온화하여 친교를 좋아하나 기호가 야비하여 관습도 아주
더럽고 악한 편이다. ( 마하라슈트라 ) 지방의 부인들은 64 종류의 동작을
모두 행하고 말투가 거칠고 성질도 난폭하며, 정사 없이 곧 (샤야나 * 성교)를
시작한다. ( 나가라 * 파타리푸트라 ) 현재의 파트나 지방의 부인들은
( 마하라슈트라 ) 와 같은 방법을 쓴다. 그러나 전자는 이러한 행위를 비밀히
하는 반면에 후자 즉 ( 마하라슈트라 ) 의 부인들은 조금도 부끄러움이
없이 태연히 행한다. ( 타미르 ) 의 부인들은 정사, 가령 포옹 등에 있어서
예민한 일이 남자에 의하여 느껴지면 성교의 예비적인 단계에 있어서도
분비액을 분비하며, 최후의 사정까지 나오는 경우도 있다.
( 바나와시카 * 곤나키의 동부지방의 부인 ) 은 성질이 온순하고
여러 종류의 정사에 능숙하다.

자기 자신의 부족한 내용을 감춰 놓고 남의 불완전한 것을
비난하고 일이 있다. 그리고 언제나 행동이 나쁜 남자를 싫어한다.
( 가우디 * 벵갈지방 ) 의 부인들은 순진하게 말을하고 애인을 사랑하며,
육체가 부드럽다. (스바르나나브하)의 기록에 의하면 그는 각 지방마다
행해지는 관습에 의해 정사에 있어서 여자가 좋아하는 것을 마쳐야 할 것이다.
라고 주장하고 있다. 그리고 혹시 그 지방의 관습이라고 하더라도 여자가
싫어하는 것은 행해서는 안된다는 것이다. 여기서 명심하지 않으면
안 되는 것은 시대 조류에 따라서한 지방의 습성이나 관습이나
의복 등의 유행이 변한다는 사실이다.

위에서 기술한 여러가지 정사중 앞의 행위가 뒤의 행위보다 한층
더 성욕열에 도움이 된다. 그러나 뒤의 것은 앞의 것에 비하여 더욱
야만적이다. 만일 남자가 여자의 뜻에 거역하고 그 반대에도 불구하고
이들 중의 어느 행위를 일방적으로 강행하면 여자는 그행위에 감당할
수 없는 것이니, 여자는 남자에게 굴복할 것은 틀림없다.
남자가 만일 ( 빈두 ) 를 여자에게 이루려고 함에 있어서, 여자는
( 마라 ) 를 하려고 하는 것으로 알거나 여자가 (마라)를 하려고 함에
대해서 남자가 ( 아브후라칸디캄 ) 을 이루려고 하면 여자는 화를
내고 반항 할 것이다.
이와 같은 사태가 생기면 치정 / 痴情 싸움이 시작된다.
여자가 한번 욕정으로 변태를 부리게 되면 남자의 머리칼을 휘어잡고
한 손으로 턱을 받치고 얼굴을 내밀어 남자의 입술을 깨물 것이며, 남자에게
파고들어 남자를 꼭 껴안을 것이다. 이리하여 술에 취한듯 흥분한 과격한
형태 때문에 남자의 몸은 많은 곳을 물리게 되는 것이다.
또는 한쪽 손을 남자의 뒤로 돌려 껴안고 남자의 가슴에 몸을 대어
다른 손으로 턱을 받치고 남자의 얼굴을 굽혀, 여자가 남자의 육체
피부에 ( 마나마라 ) 나, 그와 흡사한 여러가지의
상처 ( 깨물어서 낸 상처 ) 을 낼 것이다.

백서나 복잡한 장소 안에서 혹은 비밀히 정사를 하기 위해 정부가
여자에게서 받은 상처를 여자에게 슬며시 보인다면, 그녀는 상처를
남에게 보이지 않도록 다만 미소로서 감출 것이다. 남자가 얼굴에 미소를
짓고 추파를 던지면 여자 쪽에서는 마치 남자를 나무라듯 하면서도 전에
그 남자가 자기에게 준 정사의 상처를 슬쩍 보이면서 추파를 받을 것이다. 
이와 같이 남녀간의 애정은 오직 남녀간의 은근하고 매력적이고 수심을 주고
받게 되는 것이며 사랑과 부끄러움은 영원히 없어지지 않을 것이다.
