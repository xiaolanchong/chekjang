제3의 성 ( 트리티야 푸라크티‥·양성불구자 ) 에는 두가지 형이 있다.

1. 여자같은 여자
2. 남자같은 남자 

여자와 같은 제3의 여성이란 목소리, 거동, 감정,기호, 소심한 것,
가린한 것, 성급한 것, 부끄러움 등을 말한다. 여가의 입술로 하는 성교를
( 아우파리슈타감 * 입술로 하는 성교 ) 라고 한다.
이러한 행위에 따라 여자는 욕정의 성교의 만족을 갖고 살아갈 수 있다.
그러므로 이런 여자는 창기와 같은 행위를 하는 것이다. 이것은 곧 여자와
같은 제3의 성이다.
다음에 남자와 같은 제3의 남성은 욕정을 숨겨, 그러나 상상으로는
남자와 성교하려고 생각하고 또한 남자의 몸을 주물러 주어 삶을 가지려는
것을 말한다. 이러한 행위를 함에 있어서, 여자는 마치 껴안듯이 상대방의
몸에 기대어 다리를 안마한다. 이렇게 하여 제3의 성과 더욱 더 친해지면
다리나 사타구니에 손을 대고 심지어 남근을 주므르게 된다.
남근을 만지고 그것이 흥분하는 것을 보며 손으로 잡아 장난한다.
이와 같이 하여 꼭 제3성의 마음이 흥분하는 것을 나무라듯 하는듯이,
웃으면서 입술성교 ( 입술로 하는성교 ) 를 하자고 자기의 의사를 밝힌다.
만일 남자가 남근이 크게 발기하도록 욕정이 일어나거나 또는 생각이

이러한 방법의 성교를 하고 싶은데도 마다하고 여자가 감히
성교 하자고 말하지 않을 때는 남자가 먼저 성교 하자고 해야 한다.
이와 같이 남자가 여자에게 성교 하자고 하면 겉으로는 화가 난듯하나
여자는 결국 그 욕정에 따르게 될 것이다.

( 아우파리슈타캄 ) 의 형태에는 다옴과 같은 여덟 가지 방법이있다.


1. 니밋 타캄
2. 파르슈바트다슈탐
3. 바히산단샤
4. 안타산단샤
5. 츔비타캄
6. 파리므리슈타캄
7. 암라츄치타캄
8. 상가라


이와 같은 행위는 이에 기록한 순서에 따라서 순서대로 행해진다.
각각의 체격에 의한 행위가 끝나면, 그녀는 성교를 그만두고 싶다는 듯이
일부러 가장할 것이다. 그러면 남자는 제1의 행위를 취한 후, 제2의 행위를
하도록 이끌어 차례로 끝까지 끌고 간다. 여자가 남근을 손에 잡고 그 끝을
입술로 물고 얼굴을 좌우로 흔들면서 움직인다.
이것을 ( 니밋타캔 ) 이라고 한다. 남근의 끝을 손으로 잡고 그 아랫부분을
남기고 입술로 윗부분을 문다.
그러나 이로 물어서는 안된다. 그리고 그녀는 남자에게
애교를 보이면서 이곳까지만 하고 이 이상은 더 하지 않겠다고 한다. 
이것을 ( 파르슈바트다슈탐 ) 이라고 한다. 남자의 ( 파르슈바 ) ,
즉 남근피가 물렸다는 뜻이다.

또 그녀가 마지못해 억지로 남근의 끝을 입술로 물고 아래 위의
입술로 꼭 빨면서 남근을 잡아 뽑는다. 이러한 행위는 ( 바히산단샤 ) 라고
하는데, 이것은 남근 끝의 피를 핥기 때문에 붙여진 이름이다. 이 행위를 하고
있는 동안에 만일 남자가 좀더 많이 입안으로 삽입하라고 강요하면 그녀는
( 마니반드하 * 남근의 흠진 곳 ) 까지 근피를 아랫끝까지 끌어내리고 그 부분을
빤 다음에 침을 뱉아 버린다. 이것을 ( 안타산단샤 ) 라고 하는데, 남근의 내부가
빨라지기 때문이다. 손으로 남근을 잡고 마치 정부와 키스 하듯이
두 입술사이로 남근을 문다. 이것을 ( 춤비타캄 ) 이라고 한다.
키스하는 것과 같기 때문이다.

위에서 말한 행위, 즉 ( 츔비타캄 ) 을 행한 후 혓바닥으로 남근의 내부를
전체적으로 물었다가 전부를 빤다. 이것이 ( 파리므리슈타캄 ) 이다. 모든 내부가
혓바닥에 의해서 깨끗이 빨리기 때문이다. 이와 같이 남근의 외피를 아래로
까내리고 까진 남근을 두 번, 세 번, 네 번, 마치 ( 망고 ) 의 과실을
빨아먹듯이 빨렸다가 다시 내놓아진다. 이것을 ( 암라츄치타캄 ) 이다.
모든 이것을 ( 암라츄치타캄 : 암라는 과실 이름이고 츄치타캄은 빤다는 것 )
이라고 한다. 남자의 기호에 따라 그 혀와 입술로 남근을 빨아서 사정의
쾌감에 맛볼 때까지 계속하는 것을 ( 상가라 ) 라고 한다.
( 아우파리슈타캄 ) 의 행위에 있어서도 남자의 욕정의 정도와 그의
기교에 따라서 때리거나 울부 짖음을 하게된다.
이상으로 ( 아우파리슈타캄 ) 의 기술을 마친다

제3성자 외에, 창부·탕녀·하녀·여자 안마사 등도
이러한 성교를 행한다. 그러나 가정 부인은 이러한 행위를 해서는 안 된다.
왜냐하면 남녀간의 행위의 규정이 위반되며 또 점찮고 덕망이 있는 사람이면
이것은 비굴한 행위로 생각하기 때문이다. 또 만일 상기한 행위를 행한 후에
그 여자와 키스하면 더러운 느낌이 들기도 하고 그런 비굴한 행위를
후회하게 될 것이다. 라고도 학자 들은 말하고 있다.
그러나 창부의 경우에는 이 행위는 죄로 생각지 않는다.
그러나 예의를 저버리게 되는 다른 이유에서 이러한 행위 그들의
부인에게는 대개 하지 않는다고 ( 바짜야나 ) 는 말하고 있다.

그러나 동부 인도지방의 사람은 ( 아우파리슈타 ) 를 즐겨 행하는
관습이 있는 여자와는 성교를 하지 않는다. ( 아히슈하리카 ) 의 사람들은
창부와는 성교를 하지 않는다. 혹 성교 하더라도 입술에 키스하지 않는다.
( 사케다 ) 의 사람들은 깨끗하고 더러운 것을 차별하지 않는다.
그러므로 이들은 직접 성교하며 입술로는 성기를 핥거나 빠는 것을
상관하지 않는다. ( 나가라 * 현재의 파드나 ) 의 사람들은 창부와
성교하더라도 자기의 기분에 따라 ( 아우바캔 ) 을 그녀의 입술로
하지 않는다.
그러나 여자가 혼자 이를 좋아할 때에는 별문제다.
( 사우라세나 ) 의 사람들은 조금도 부끄럼 없이 이를 행한다.
그들은 스스로 속이기를 여자의 본능은 이미 더러운 성품이 들어
있을 뿐만 아니라 사상도 나쁘며, 따라서 본래부터 여자가
깨끗하다거나 품행이 단정하다거나 또는 약속을 잘지키고 신임할 수
있다는 것을 맞지 않는 말이다. 그러나 여자는 ( 푸르샤알트하 ) ,
즉 ( 카마 * 성애 ) 의 갖기에는 없어서는 안 될 절대 필요한 상대이다.

그러므로 여자가 깨끗하다는 것은 오직법전의 규정에만 있을 뿐이다.
이 사실을 증거키 위하여 다음의 한 글을 인용하겠다.
( 송아지의 입은 보통때에는 깨끗하지 않으나 어미소로부터 젖을 빨아먹을
때에는 깨끗하다. 개의 입은 사냥할 때에는 깨끗하다. 새의 부리는
나무에서 과실을 쪼아 떨어뜨릴때 깨끗하고 여자의 입은 성교 할 때와
키스를 할 때는 깨끗하다. )

어느 남자가 정력이 줄어들거나, 늙었거나 뚱뚱하거나, 또는
다른 결과로 남에게 만족을 줄 수 없으면서도 때때로 성교의 욕정이
일어나서, 그의 성기를 자극하여 사정함으로 쾌감을 상상하고 성교를
하고자 할 때는 아름다운 귀걸이를 단 소년을 고용하여 ( 아우파리슈타캄 ) 을
하게 된다. 이들 소년은 주인의 남근을 입안에 삽입하고 그의 욕구를
만족시켜 준다.
이런 예에 관하여는 의견의 차이가 많으며, 위에 쓴 ( 스므리티 ) 의
문전 / 文典 도 전혀 다른 의견으로 해석하고 있다.
예를 들면 ( 스트리 ) 를 단순히 처라고만 해석하듯이... 그러나 남자는
지방의 관습과 자신의 진심과 신앙에 따라 움직인다.
라고 ( 바쨔야나 ) 는 말하고 있다.

어떤 두 남자가 도시생활에서 이러한 행위에 의해서 서로
문안하는 것을 좋아하는 습관이 있고, 서로가 깊이 친분이있을 경우에는
서로 ( 아우파리슈타캄 ) 으로 사정하는 쾌감을 느낀다. 또한 남자가 여자에게
이러한 수법을 쓸 때가 있다. 즉 남자가 여자의 음부에 ( 아우파리슈타캄 ) 의
행위를 하는 것이다. 그 수법은 여자의 입술에 키스하는 것과 같다.
남녀가 동시에 이를 행할 경우가 있다. 남녀가 서로 꺼꾸로 누워, 서로의
성기를 입으로 잡아서 행한다. 이 행위를 ( 카키라 ) 라고 한다.
그 상태가 마치 까마귀가 더러운 물건을 입으로 쪼아먹는
것과 같기 때문이다.

이러한 뜻으로 해서 창기들이 노예, 혹은 코끼리 모리꾼과 같은
천한 사람과 사랑에 빠져 어질고 선량한 사람을 몰라보는 수가 있다.
학덕있는 파라문 / 婆羅門 , 재상 , 명예로운 벼슬에 있는 사람들은 절대로
( 아우파리슈타캄 ) 을 자신이 직접 해서는 안된다. 또 타인을 상대로
해서는 안 된다. 법전이 이러한 행위를 하지 말라고 정했다고 해서가 아니라,
그것은 꼭 행하지 않으면 안 되는 것은 아니기 때문이다. 물건을 상의없이
규정한 것이 법전이며, 그 사용은 쓸 필요에 따라서, 단지 필요한 이에게
한하여 사용하게 되어 있다.

개고기가 정력을 증진시키고 몸에 좋다고 하여, 이를 약으로 먹어도
의서 / 醫書 에서는 좋다 하더라도 총명한 사람이 어찌 이것을 먹겠는가.
그러나 그렇지가 않다. 어떤 부류에 속한 사람들이 이러한 행동을 하는 것은
확실하다. 또 이와 같은 행위가 보편적으로 행해지는 지방이 있다. 때로는
이와 같은 행위도 필요할 경우가 있는 것이다. 따라서 사람은 우선 위치와
시간과 그 수법과 법전의 규정과 자기의 위치 등을 생각하여 이들의
행동을 행할 것인지 아닌지를 확정해야 할 것이다.
( 아우파리슈타캄 ) 의 행위는 극비로 하여 또 남녀의 본능은
모름지기 욕정적이기 때문에 이러한 동작은 언제, 어떻게 왜 하는
것인가에 대하여는 아무도 이를 알 수 없다.
